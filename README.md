# DARA Store
An ecommerce website with backend implemented in nodejs and expressjs and database hosted on mysql local server. It contains features like product listing(only by admin), user login/signup, admin login/signup, cart checkout, reviews system etc. We have made this as a part of DBMS academic group project and I contribution includes the backend of this website.
